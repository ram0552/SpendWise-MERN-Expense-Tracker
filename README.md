# SpendWise

SpendWise is a full-stack MERN expense tracking and budgeting application.

## Features

- JWT Authentication
- Expense Tracking
- Income Management
- Budget Tracking
- Analytics Dashboard
- Charts & Reports
- CSV Export
- Responsive UI

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Zustand
- Axios
- Recharts

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Installation

### Backend

```bash
cd backend
npm install
